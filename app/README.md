# ShowsApp

- Angular 5
- Jasmine
- Karma
- SASS
- Typescript 2
- Webpack 3


#### Features
- Provides an application to help sellers and buyers exchange goods



Getting Started
---------------

#### Prerequisites
- `node >= 6`
- `npm >= 5`

#### Quick Start

```shell
$ npm install
$ npm start
```
 - Go to 'http://localhost:5550/'

Application Build
-----------------

```shell
$ npm run clean
$ npm run build:prod
```
